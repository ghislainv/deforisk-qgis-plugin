Conseils pour les grandes juridictions#

Quelques conseils pour les grandes juridictions#

Il peut être nécessaire de modifier les paramètres par défaut du plugin pour les grandes juridictions (par exemple, à l’échelle d’un pays).

  1. Utilisez une distance maximale à la lisière de la forêt (Max. distance to forest edge (m)) plus grande, par exemple 6000 m dans les modèles Benchmark et MW models.

  2. Si vous souhaitez économiser du temps de calcul et consulter les résultats les plus importants, vous n’avez qu’à :

    • Calibrez les modèles sur la période de calibration, en cochant calib. period dans la case Fit model to data.

    • Prédire pour la période de validation, en cochant t2 validation dans la case Predict the deforestation risk.

  3. Pour les modèles FAR :

    • Utilisez une taille de cellule spatiale Spatial cell size (km) plus grande pour l’échantillonnage, par exemple 10 km pour éviter d’estimer un trop grand nombre de paramètres pour les effets aléatoires spatiaux.

    • Utilisez une taille de cellule spatiale pour l’interpolation Spatial cell size interpolation (km) plus grande, par exemple 1 km.

  4. Pour les modèles MW, n’utilisez qu’une seule taille de fenêtre Window sizes (# pixels), par exemple 21. Cela correspond à 630 m pour une résolution de 30 m.

  5. Pour l’étape de validation, utilisez une seule taille de cellule pour la grille grossière Coarse grid cell size (# pixels) et augmentez sa taille, en utilisant par exemple 300. Cela correspond à 9 km pour une résolution de 30 m.
