Plugin QGIS deforisk#

Idiomas#

El sitio web del plugin deforisk para QGIS está disponible en inglés o español .

Objetivo#

El plugin QGIS deforisk puede utilizarse para cartografiar el riesgo de deforestación de un país o área de interés. Se pueden utilizar cuatro modelos para obtener los mapas de riesgo: los modelos iCAR, GLM, Random Forest y Moving Window. Estos cuatro modelos pueden estar comparados con un modelo de referencia que asume una simple disminución del riesgo de deforestación con la distancia al borde del bosque. Todos los modelos se calibran utilizando observaciones de deforestación pasadas para un periodo de tiempo determinado entre 2000 y 2022. Los mapas de cambio de la cubierta forestal pueden ser proporcionados por el usuario o derivados de dos productos de cambio global de la cubierta arbórea/forestal: Global Forest Change y Tropical Moist Forests.

Los mapas de riesgo de deforestación obtenidos mediante este plugin pueden utilizados para estimar el impacto de la reducción de emisiones de los proyectos de conservación forestal dentro del VCS Jurisdictional and Nested REDD+ Framework.

Especificidades#

Basodo sobre Python. El plugin deforisk se basa en cuatro paquetes de Python desarrollados específicamente para modelar la deforestación: geefcc, pywdpa, forestatrisk y riskmapjnr. El paquete geefcc puede utilizarse para elaborar mapas de cambio de la cubierta forestal a partir de Google Earth Engine (GEE) y descargarlos localmente utilizando dos productos de cambio global de la cubierta arbórea/forestal: Global Forest Change o Tropical Moist Forests. El paquete pywdpa permite descargar archivos vectoriales de áreas protegidas de cualquier país utilizando la Base de Datos Mundial sobre Áreas Protegidas (WDPA). El paquete forestatrisk proporciona funciones para modelizar la deforestación y predecir el riesgo espacial de deforestación utilizando diversas variables explicativas (distancia al borde del bosque, elevación, áreas protegidas, etc.) y diversos modelos estadísticos, incluidos los modelos iCAR, GLM y Random Forest. El paquete riskmapjnr permite derivar mapas de riesgo de deforestación siguiendo las metodologías JNR de Verra, que incluyen un modelo de ventana móvil y un modelo de referencia que asume una disminución del riesgo de deforestación con la distancia al borde del bosque.

Procesando rásters por bloques. Los archivos raster de los cambios en la cubierta forestal y las variables explicativas pueden ocupar varios gigabytes en disco. El procesamiento en memoria de rásters tan grandes puede ser imposible en ordenadores con RAM limitada. Las funciones utilizadas en el plugin deforisk procesan grandes rásters por bloques de píxeles que representan subconjuntos de los datos ráster. Esto hace que el cálculo sea eficiente, con un bajo uso de memoria. La lectura y escritura de subconjuntos de datos ráster se realiza utilizando funciones de GDAL, una dependencia del plugin. Los cálculos numéricos sobre matrices se realizan con el paquete NumPy de Python, cuyo núcleo está formado principalmente por código C optimizado y compilado que se ejecuta rápidamente.

Ejecutar tareas en paralelo. El enfoque más avanzado para seleccionar el mejor mapa de riesgo de deforestación y predecir la deforestación implica comparar varios modelos, ajustar los modelos utilizando el cambio de la cubierta forestal en diferentes periodos de tiempo y predecir el riesgo de deforestación en varias fechas. Esto implica repetir un elevado número de tareas. Para ahorrar tiempo de cálculo, el plugin deforisk utiliza el gestor de tareas de QGIS que permite ejecutar varios análisis en paralelo.

Independiente del OS. Utilizando tanto el cálculo por bloques para grandes rásters como la paralelización de tareas, el plugin deforisk permite seleccionar el mejor mapa de riesgo de deforestación y predecir la deforestación para grandes países o áreas de interés en un tiempo limitado, incluso en ordenadores personales con hardware de rendimiento medio. Dado que deforisk es un plugin de QGIS escrito en Python, debería funcionar en todos los sistemas operativos capaces de ejecutar QGIS, incluyendo Windows (\(\geq10\)), Linux y Mac OS.

Instalación del plugin deforisk en QGIS#

Nota

Dependencias: QGIS y GDAL deben estar instaladas en su sistema antes de utilizar el plugin deforisk. En sistemas Unix, también debe instalar osmconvert y osmfilter. En sistemas Windows, estas dependencias ya están incluidas en el plugin como archivos binarios .exe, por lo que no es necesario instalarlas. A continuación, los paquetes de Python forestatrisk y riskmapjnr deben estar instalados en tu sistema. Siga las instrucciones de instalación para instalar estas dependencias.

  • Descarga el archivo deforisk zip de GitHub.

  • Abra QGIS.

  • En la barra de menú de QGIS, vaya a Extensiones/Instalar extensiones/Instalar desde ZIP.

  • Seleccione el archivo zip que se ha descargado.
